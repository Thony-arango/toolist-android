# Política de Privacidad — TooList

**Última actualización:** abril de 2026

## 1. Información general

TooList es una aplicación Android de listas de compras desarrollada como proyecto académico en la Universidad Pontificia Bolivariana (UPB), 2026. Esta política describe cómo se recopila, usa y protege la información de los usuarios.

## 2. Datos que recopilamos

### 2.1 Cuenta de usuario (Firebase Authentication)
- Correo electrónico y contraseña (gestionados por Google Firebase Authentication)
- No almacenamos contraseñas en texto plano; Firebase las maneja de forma segura

### 2.2 Datos de uso (Cloud Firestore)
- Listas de compras creadas por el usuario (nombre, color, fecha)
- Productos agregados a las listas (nombre, cantidad, precio estimado, categoría, estado)
- Categorías personalizadas creadas por el usuario

### 2.3 Datos que NO recopilamos
- Ubicación geográfica
- Contactos del dispositivo
- Información de pago
- Datos biométricos

## 3. Uso de la información

Los datos recopilados se usan exclusivamente para:
- Permitir al usuario crear y gestionar sus listas de compras
- Sincronizar los datos entre sesiones del mismo usuario
- Mantener el historial de listas y productos

Los datos **no se comparten** con terceros ni se usan con fines publicitarios o comerciales.

## 4. Almacenamiento y seguridad

Los datos se almacenan en **Google Cloud Firestore** bajo el proyecto Firebase `app-movil-toolist`. Google aplica medidas de seguridad estándar de la industria. Cada usuario solo puede acceder a sus propios datos gracias a las reglas de seguridad de Firestore.

## 5. Retención de datos

Los datos se conservan mientras la cuenta del usuario esté activa. El usuario puede solicitar la eliminación de su cuenta y todos sus datos asociados contactando al desarrollador.

## 6. Derechos del usuario

El usuario tiene derecho a:
- Acceder a sus datos en cualquier momento desde la aplicación
- Solicitar la corrección de datos incorrectos
- Solicitar la eliminación de su cuenta y datos

## 7. Cambios en esta política

Esta política puede actualizarse ocasionalmente. Se notificará a los usuarios sobre cambios significativos a través de la aplicación.
